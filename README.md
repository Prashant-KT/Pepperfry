<h1>Pepperfry</h1>
 Pepperfry is an e-commerce website where users can purchase various kinds of furnitures and home decors.
<h3>Features:</h3>
<ul>
<li>Sign-in/Sign-up</li>
  <li>Products page</li>
  <li>Wishlist page</li>
  <li>Payment page</li>
<li>User authentication</li>
<li>Sorting/filter functionality</li>
  </ul>

<h3>Tech Stack:</h3>
